# Superset Setup

Connect Apache Superset to the same Render PostgreSQL database used by the API.

Useful datasets:

- `superset_question_progress`
- `superset_pattern_summary`
- `superset_daily_study`

Suggested charts:

- Big number: solved questions
- Bar chart: solved percent by pattern
- Table: revise queue by pattern and title
- Time series bar: study minutes by day and focus
